# Ableton Live 12

**Browser Tagging & Search Improvements**

## Overview

Live 12 introduces improved tagging and search functionality to help users quickly find sounds, presets, and devices.

These updates streamline browsing and improve workflow efficiency.

---

## New Features

### Custom Tags

Users can now create custom tags for:

* Samples
* Presets
* Devices
* Clips

To create a custom tag:

1. Right-click an item
2. Select **Edit Tags**
3. Add tag
4. Press Enter

Custom tags appear in the Browser sidebar.

---

### Tag Filtering

Tags can now be combined to refine searches.

Example:

* Drum + Analog + Warm

Live displays results matching all selected tags.

---

### Improved Search

Search now includes:

* Fuzzy matching
* Faster results
* Tag-aware search

Search results update in real-time.

---

## Workflow Example

To find a bass preset:

1. Open Browser
2. Click **Bass**
3. Add tag **Analog**
4. Add tag **Warm**

Live filters results instantly.

---

## Tips

* Use custom tags for projects
* Tag frequently used presets
* Combine tags for precise results
