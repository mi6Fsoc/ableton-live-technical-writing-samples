# Roar (Saturation & Color)

**Technical Documentation Sample**

## Overview

Roar is a multistage saturation and coloring device that allows you to shape harmonics, add character, and enhance dynamics. It combines multiple saturation stages with routing options and modulation capabilities.

Roar can be used for subtle warmth, aggressive distortion, or dynamic tonal shaping.

---

## Adding Roar

To add Roar:

1. Open the Browser
2. Navigate to **Audio Effects**
3. Drag **Roar** onto an audio or MIDI track

Roar can also be added to return tracks and groups.

---

## Interface Overview

Roar consists of:

* Three Saturation Stages
* Routing Section
* Tone Controls
* Modulation Section
* Output Controls

Each stage can be enabled or disabled independently.

---

## Saturation Stages

Roar includes three processing stages:

### Stage A

Primary saturation stage for shaping harmonic content.

**Controls**

* Drive — Amount of saturation
* Tone — Frequency emphasis
* Mix — Dry/Wet balance

---

### Stage B

Secondary processing stage for additional coloration.

**Controls**

* Drive
* Bias
* Character

---

### Stage C

Final stage for refinement and shaping.

**Controls**

* Drive
* Width
* Output

---

## Routing Modes

Roar supports multiple routing configurations:

### Serial

Stages process audio sequentially.

Use this mode for:

* Heavy saturation
* Progressive tonal shaping

---

### Parallel

Stages process audio simultaneously.

Use this mode for:

* Subtle harmonic layering
* Parallel distortion

---

### Mid/Side

Processes mid and side signals independently.

Use this mode for:

* Stereo enhancement
* Width control

---

## Modulation

Roar includes modulation sources:

* LFO
* Envelope Follower
* Macro Controls

To assign modulation:

1. Click **Modulation Mode**
2. Select a modulation source
3. Drag to parameter
4. Adjust modulation amount

---

## Use Cases

### Subtle Warmth

* Low Drive
* Serial routing
* Mix below 50%

### Aggressive Distortion

* High Drive
* Multiple stages enabled
* Parallel routing

### Stereo Enhancement

* Mid/Side mode
* Moderate Drive on Side channel

---

## Tips

* Use parallel routing to maintain clarity
* Combine with EQ for tonal shaping
* Automate Drive for dynamic effects

---

## Notes

* High Drive settings may increase output level
* Use Output control to compensate gain
